# NebulaGraph Studio 版本更新说明

- 功能
  - 支持查看 Schema 的[创建语句](../../nebula-studio/manage-schema/st-ug-view-schema.md)。
  - 新增产品反馈页面。

- 优化
  - 优化慢查询，取消超时限制。
  - 浏览器兼容提示。
  - 优化登录页。
  - 控制台支持使用`#`添加注释。
  - 优化控制台界面。

- 缺陷修复
  - 修复上传文件后列表不刷新的问题。
  - 修复 Schema 草图中无效的错误提示。
  - 修复切换登录用户后，**查看 Schema** 数据未清除的问题。
  - 修复 Schema 草图的缩略图展示的问题。
